# LSTM-vs-Reservoir
This is an effort to prove that Reservoir Computing is more efficient and precise than LSTM for training and inference in time-series environmental data.

We'll abbreviate Reservoir Computing as RC throughout.

Main libraries used for LSTM:
Pytorch
Pytorch Lightning

Main libraries used for RC:
ReservoirPy

MLflow is used for hyperparameter optimization in both programs.
